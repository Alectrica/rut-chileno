rut-chileno
===========

ruby gem to validate and format chilean R.U.T.

instalar como gema ruby
```
	$ gem install rut_chileno
```
o en Ruby on Rails, en el Gemfile
```
	gem 'rut_chileno'
```
luego
```
	$ bundle install
```
